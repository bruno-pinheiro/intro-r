Aula 02: Introdução ao tidyverse

Tópico 1

  1. Missing data: o “dado faltante” é um dado que deveria ou poderia estar presente no conjunto de dados, mas por algum motivo não está. No R os valores ausentes são geralmente dados por NA.

Agora vamos uní-los em um data.frame (mais adiante explicaremos o que é um data frame), para deixar os dados em forma de tabela.

Como pode ver, cada vetor se tornou uma variável (coluna) da nossa tabela df. Agora sim vamos realizar algumas operações usando as duas variáveis, para ver o que acontece. Para chamar uma variável dentro de um data frame utilizamos o operador cifrão $ entre o nome do objeto e da variável: objeto$variavel. Vejamos abaixo:

# df$v1 + df$v2

Além disso, nos exemplos acima nós extraímos a cabeça e a rabeta do dataset inteiro, mas muitas vezes queremos fazer esta extração apenas para uma variável. Neste caso vamos utilizar a mesma sintaxe da função usada acima, porém vamos especificar qual variável queremos acessar.

Uma forma básica de acessar variáveis em um data frame no R é com a utilização do sinal de cifrão como operador $ (conhecido como dolar sign) em inglês. Basta colocar o operador após o nome de algum objeto e então especificar uma variável. Ou seja, às esquerda o nome da objeto, ao centro o operador e à direita o nome da variável, assim: objeto$variavel.

Analise o exemplo a seguir:

# head(df$Sexo)

O resultado visto acima apresenta apenas os seis primeiros valores apenas da variável Sexo e não de todo o conjunto de dados, como nos exemplos anteriores.

Tente fazer abaixo para as demais variáveis do dataset, trocando “__" pelo nome das colunas.

  • Dica: se não lembra o nome das demais variáveis, você pode consultá-las usando a função names(nome-do-objeto).
# head(df$__)

Agora sim, dado que nem no começo nem no fim encontramos algum problema aparente, podemos seguir adiante.

Agora que os dados foram importados é preciso saber se está tudo certo. É sempre bom saber se não houve nenhum erro durante a importação.
