Primeiros comandos

Primeiros comandos

Importar dados .csv

dados <- read.csv()

Checar dados

  • Nomes de variáveis
names()
  • Estrutura dos dados
str()
  • Sumário descritivo
summary()
  • Cruzamento de variáveis numéricas
table()
prop.table(table())
  • Visualização de variáveis numéricas (histograma e boxplot)
hist()
boxplot()
  • Relação entre variáveis numéricas (scatterplot)
plot()
  • Visualização de variáveis categóricas (barplot)
barplot()
barplot(table())

Uso de pacotes

install.packages("ggplot2")
